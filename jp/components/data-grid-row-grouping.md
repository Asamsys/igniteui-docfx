## Angular テーブル列グループ化

Angular Data Table / Data Grid を使用すると、列の値を「固定ヘッダー」行グループにまとめることができます。この機能は Microsoft Outlook の Group By 機能同様、独自の基準に基づいてデータを視覚的にグループ化する簡単な方法を使用します。
