## 高パフォーマンス Angular Table

Angular Data Table / Data Grid は、さまざまなデータ シナリオのパフォーマンスに最適化されています。速いロード時間、タイムラグや画面のちらつきのないスムーズなスクロール、グリッドの列と行の仮想化により、Angular グリッド アプリケーション内の行と列を無制限にスクロールできます。
